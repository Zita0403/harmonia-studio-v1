# Vizsgamunka - Harmónia Stúdió

![Harmónia Stúdió kezdőlap képernyőképe](assets/images/cosmetic_website_v1.png)

Ez a weboldal egy **fiktív kozmetikai szalon** bemutatóoldala, amelyet **első vizsgamunkaként** készítettem el a **Full Stack Webfejlesztő képzés** első moduljának (HTML+CSS(+JS alapok) front-end webfejlesztő tanfolyam) végén.

## Leírás

A vizsgamunka célja egy **modern, reszponzív és szemantikailag helyes weboldal** készítése, amely több aloldalt is tartalmaz és bemutatja a **HTML, CSS, SASS, JavaScript, jQuery** alapismereteimet.
A weboldal **mobilra, tabletre és desktop nézetre** is optimalizált.
A reszponzivitást **SASS media query-k, mixinek és rugalmas CSS layout-ok (Grid, Flexbox)** biztosítják.
A HTML és CSS kódot W3C Validator [https://validator.w3.org/] segítségével ellenőriztem.

## Felépítés

A weboldal az alábbi oldalakból áll:
- **index.html** - Kezdőlap, rövid bemutatkozás, fő tartalmak
- **facial-treatment.html** - Arckezelések ismertetése
- **body-treatment.html** - Testkezelések ismertetése
- **facial-treatment.html** - Szőreltávolítási eljárások ismertetése
- **make-up.html** - Sminkkészítési szolgáltatások
- **price-list.html** - Árlista táblázatos formában
- **booking.html** Időpontfoglaló űrlap

## Használt technológiák

- **HTML5** - szemantikus szerkezet
- **CSS3** - SASS-ból gernerálva, reszponzív design (Grid, Flexbox, media query-k)
- **SASS (SCSS)** - moduláris, jól karbantartható stíluslapok, változók és mixinek használata
- **JavaScript (ES6)** - interaktív funkciók használata
- **jQuery** - DOM-kezelés és animációk egyszerűsítése
- **Google Fonts + Font Awesome** -  ikonok használata, esztétikus megjelenés

## Tartalmi elemek

- **Képek:** Saját készítésű favicon és szabad felhasználású forrásból származő képek Pexels [https://validator.w3.org/] oldalról.
- **Űrlap:** Időpontfoglalási form, mezőellenőrzéssel.
- **Táblázat:** Árlista a szolgáltatásokról
- **Menü:** Reszponzív navigációs sáv
